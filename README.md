# Workflow
Khi làm tính năng mới, ví dụ AES file encryption:

1. Chuyển sang dev:

git checkout dev
git pull

2. Tạo feature branch:

git checkout -b feature/aes-file-encryption

3. Code

4. Commit:

git add .
git commit -m "Implement AES file encryption"

5. Push:

git push -u origin feature/aes-file-encryption

6. Lên GitHub → Compare & Pull Request

Base: dev

Compare: feature/aes-file-encryption

Điền theo PR template

Create PR

7. Đợi review → Merge vào dev
